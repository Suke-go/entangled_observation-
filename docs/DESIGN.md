# ±Quantum: 量子インスタレーション設計指針

## 概要

+（プラス）と −（マイナス）のみで構成されるモーショングラフィクスによるメディアアート作品。バックエンドに量子計算シミュレーションを持ち、カメラベースのアイトラッキングによる観客の注視を「量子測定」として系に反映する。同一空間に設置された二台のディスプレイがサーバーを介してエンタングルメント状態を共有し、一方の観客の視線が他方のディスプレイに非局所的影響を及ぼす。

美学的参照: 江渡浩一郎「+/- Minimal VJ」(http://eto.com/)
概念的参照: 落合陽一「リキッドユニバース」シリーズ（画像の連続的変容）

## 1. 理論的基盤

### 1.1 量子ビットと ± の対応

- パウリZ基底の固有値は +1 と −1 であり、`|+1⟩ = |0⟩`, `|−1⟩ = |1⟩` に対応する
- パウリX基底（Hadamard基底）では `|+⟩ = (|0⟩ + |1⟩)/√2`, `|−⟩ = (|0⟩ − |1⟩)/√2`
- + は「建設的重ね合わせ」、− は「破壊的重ね合わせ」を意味し、符号が位相情報を担う

### 1.2 核心となる量子概念

| 概念 | 数学的記述 | 視覚的対応 |
|------|-----------|-----------|
| 重ね合わせ | `α\|+⟩ + β\|−⟩`, `\|α\|² + \|β\|² = 1` | +/- の確率的揺らぎ（形態的曖昧性） |
| 位相 | α, β が複素数 | 揺らぎのリズム・タイミング |
| 射影測定 | `P_k\|ψ⟩ → \|k⟩` with prob `\|⟨k\|ψ⟩\|²` | 注視による +/- の確定 |
| エンタングルメント | `\|Φ+⟩ = (\|++⟩ + \|−−⟩)/√2` | 二台のディスプレイ間の非局所的相関 |
| デコヒーレンス | 環境との相互作用による量子性の喪失 | 揺らぎ→ランダムノイズへの退化 |
| 量子ゲート（Hadamard） | `H = (1/√2)[[1,1],[1,−1]]` | 確定状態→揺らぎの開始 |
| CNOT | 制御ビット `\|1⟩` のとき標的を反転 | ディスプレイ間ペアリングの生成 |

### 1.3 ベクトル空間性

- 古典的理解: +/- は離散集合 `{+, −}` の要素
- 量子的理解: +/- は 2次元複素ベクトル空間 `C²` の正規直交基底
- 任意の状態は線形結合 `α|+⟩ + β|−⟩` として連続的に存在（ブロッホ球 S²）
- **「足し合わせることができる」という線形性そのものが、古典的二値論理にはない構造**

### 1.4 直観に反すること（作品のメタメッセージの根幹）

1. **見ることが対象を変える**: 観測が状態を確定させる（射影測定の不可逆性）
2. **位相は見えないが存在する**: 複素確率振幅の角度情報は直接観測できないが干渉を通じて効果を持つ
3. **基底の選び方で「同じ」と「違う」が変わる**: 計算基底とHadamard基底で異なる測定結果分布
4. **部分は全体を記述できない**: エンタングルした系の部分系は混合状態（情報が相関の中にだけ存在）
5. **相関は事前に決まっていたのではない**: ベルの定理 — 隠れた変数理論の否定

## 2. メタ的メッセージ

### 2.1 中心命題

**「見るとはどういうことか」**

この作品が問いかけるのは「何が見えるか」ではなく、観察行為そのものについての問い。

### 2.2 メッセージ階層

- **表象の限界**: +/- は人間が世界を分節化するための最小単位。量子力学はその分節化以前の「何か」の存在を示すが、我々はそれを直接表象できない。パフォーマンスは「表象不可能なものを表象しようとする試み」であり、その構造的な失敗がメッセージになる
- **観察者が世界を構成する**: 観客が見なければ +/- は確定せず、見方（注視/ぼんやり）によって異なる記号が生成される。これは哲学的主張ではなく、システムの物理的構造として実装される
- **分離不可能性**: 二台のディスプレイの体験は独立ではない。自分の画面が自分の制御を超えた振る舞いをしていることで、「他者の存在が自分の世界を構成している」ことが体験される

### 2.3 リキッドユニバースとの差異

- 落合作品: 生成AIによる画像変容。観客は「受け取る」立場
- 本作品: 量子シミュレーション＋視線による画像変容。観客が不可分に介入する
- 落合作品: LEDビジョン上のフルカラー映像
- 本作品: +/- のモノスペーステキスト。記号と図像の間の緊張関係

## 3. 鑑賞体験と量子構造の対応

### 3.1 視線＝射影測定の同型性

| 鑑賞行為 | 量子力学的対応 |
|----------|---------------|
| 近づいて個々の +/- を読む | 位置基底での強い射影測定 |
| ぼーっと遠くから見る | 粗視化 (coarse-graining)、古典極限 |
| 視線を移動させる | observable の選択 |
| 凝視し続ける | 量子ゼノ効果（頻繁な測定が遷移を凍結） |
| 素早く視線を動かす | 広範囲の急速な崩壊 |
| 目を閉じる/視線を失う | 測定の不在→重ね合わせへの回帰 |

### 3.2 距離と人数の効果

- **近距離・少人数**: 個々の +/- を識別。狭い領域のみ崩壊。全体像は見えにくい
- **遠距離・少人数**: 密度テクスチャとして知覚。画像全体が見える条件だが測定精度は粗い（弱測定）
- **近距離・多人数**: 複数視線が異なる領域を同時崩壊。画像が急速に「現像」される
- **多人数＋二台**: エンタングルメントにより、A側の視線がB側の画像現像にも寄与

### 3.3 スケール階層

| スケール | 知覚 | 数学的構造 |
|---------|------|-----------|
| ミクロ（記号レベル） | 個々の +/- を読む | 離散集合 `{+, −}` |
| メソ（パターンレベル） | テクスチャ・干渉・モアレ | ベクトル空間 C² の連続構造 |
| マクロ（全体レベル） | 濃淡による図像（海など） | 古典極限。統計的平均値のみ残る |

## 4. 視覚設計

### 4.1 美学原則（江渡 +/- Minimal VJ 参照）

- **モノスペース厳守**: 等幅フォントのグリッド。格子構造が画面全体を支配
- **+/- 以外の記号は使用しない**: 記号の種類ではなく配置・リズム・密度で表現
- **格子からの逸脱は一切なし**: 規則性があるからこそパターンの変化が意味を持つ
- **無音**: 視覚だけで完結する潔さ

### 4.2 離散測定と形態的曖昧性

各セルは常に + か − のどちらかである。中間はない。これが量子測定の離散性を反映する。

実装方式: **毎フレーム確率的二値選択（離散フリッカー）**

- 毎フレーム、各セルは pPlus の確率で + を、(1 - pPlus) の確率で − を描画
- 横棒（−の構成要素）は常に描画、縦棒（+の一画）は確率的に出現/消失
- 高速に +/− が切り替わり、人の目にはどちらとも確定できない「揺らぎ」として知覚される
- **測定 = 揺らぎが止まる**。一つの値にロックされ、静止する。これが崩壊

**コヒーレンスによる揺らぎの質の制御** (密度行列の非対角要素 `|⟨+|ρ_i|−⟩|`):

- コヒーレンス大: 規則的なリズムの切替（sin波的）→「積極的に両方である」量子的重ね合わせ
- コヒーレンス小: 不規則なランダム切替 →「ただ分からない」古典的混合

### 4.3 +/- 密度差による構造の可視化

- + は十字（画素密度高 → 暗く見える）
- − は横線のみ（画素密度低 → 明るく見える）
- 波の密度が高い場所 → + に崩壊しやすい → + が密集 → 暗い帯
- 波の密度が低い場所 → − に崩壊しやすい → − ばかり → 明るい帯
- **干渉縞が +/− の離散格子パターンとして出現する**

### 4.4 確定/未確定の視覚的コントラスト — 干渉縞の出現

- 未確定: 全セルが +/− に高速フリッカー。構造は知覚できない
- 確定: 揺らぎが止まり静止。+/− の配置が確定し固定される
- **干渉縞が現れる**: 量子ウォークの波密度が +/− のバイアスを決めているため、確定した配置は波の干渉構造を反映する
  - 建設的干渉（波密度大）→ + が密集 → 暗い帯
  - 破壊的干渉（波密度小）→ − だけ → 明るい帯
  - **観測するまで構造は存在しない。観測が構造を確定させる**
- 画面上に確定の「島」と揺らぎの「海」が共存
- エンタングルメントによる「遠隔の確定」: 自分が見ていない場所が突然静止し、構造の一部が出現する

### 4.5 関連する物理現象

| 現象 | 対応 | 意味 |
|------|------|------|
| クラドニ図形 | 量子ウォーク=振動板、測定=砂をまく | 「見る」行為が模様を確定させる |
| 二重スリット | 崩壊=粒子検出、集積=干渉縞 | 離散的測定の集積が波動構造を明らかに |
| イジング模型 | フリッカー=高温無秩序、崩壊=低温秩序 | 観測が「冷却」として構造を結晶化 |
| 対称性の自発的破れ | 個々の+/-は偶然、全体の構造は必然 | 同じ干渉縞でも崩壊のたびに異なる配列 |
| デコヒーレンス | フリッカー(波動)→確定(+/-粒子) | 観測者が波動-粒子の境界を決定 |

## 5. システムアーキテクチャ

### 5.1 全体構成

```
┌─────────────┐                                      ┌─────────────┐
│ Display A   │          ┌──────────────┐             │ Display B   │
│ + Webcam    │◄──ws────►│  Server      │◄────ws─────►│ + Webcam    │
│ + Browser   │          │  (relay)     │             │ + Browser   │
│             │          │  崩壊イベント │             │             │
│ L1: QW波紋  │          │  の中継のみ   │             │ L1: QW波紋  │
│ L2: 量子回路│          └──────────────┘             │ L2: 量子回路│
│ Eye Track   │                                      │ Eye Track   │
│ Renderer    │                                      │ Renderer    │
└─────────────┘                                      └─────────────┘

 クライアントが L1+L2 を自前で実行。
 サーバーは崩壊イベントの A↔B リレーのみ。
 利点: 低レイテンシ、オフライン単体動作可能。
```

### 5.2 クライアントサイド量子シミュレーション

全ての量子シミュレーションはブラウザ内で実行される。サーバーは崩壊イベントのリレーのみ。

#### 5.2.1 L1: 量子ウォーク（quantum-walk.js）

- **離散時間量子ウォーク**: 2D格子上のHadamardコイン
- グリッド解像度（例: 130×65セル）の `Float32Array` に波動関数の確率密度を保持
- フリッカー: 各セルの `+`/`−` 切替確率がウォーク密度に依存
- 注視点（視線/マウス座標）付近に波束を注入 → 干渉パターンが観察者の視線から拡散

#### 5.2.2 L2: タイルシステム（quantum-tile.js）

- 画面を 32×16 タイル（512個）に分割、各タイル k=2 量子ビット
- 状態ベクトル `|ψ⟩ ∈ C^4` でタイルレベルの重ね合わせ管理
- 注視点からガウシアン距離 (σ=3 タイル) に基づく射影測定
- 崩壊 → 再量子化（5秒後にHadamardゲートで重ね合わせに回帰）

#### 5.2.3 崩壊パターン（collapse-patterns.js）

- 3つの白黒画像（干渉縞・眼のシルエット・猫のシルエット）を輝度マップとして読み込み
- タイル崩壊時、セルレベルで輝度に基づいて `+`/`−` を決定
  - 暗ピクセル → `+` 確率高（十字 = インク密度高 = 暗）
  - 明ピクセル → `−` 確率高（横棒のみ = インク密度低 = 明）
- 20秒サイクルでパターンが自動切替 → 観測するたびに異なる像が出現

#### 5.2.4 測定の物理的正当化

- 注視点 `(x, y)` からの距離に応じたガウシアン重みづけ
- 近い → 強い射影測定（ほぼ確定）
- 遠い → 弱測定（POVM近似、状態を少しだけ更新）
- WebGazer.jsの精度（~100ピクセル）が自然にPOVMの「ぼやけ」を提供

### 5.3 アイトラッキング（gaze-tracker.js）

- **WebGazer.js v3.5.3**: MediaPipe FaceMesh → 虹彩位置 → リッジ回帰でスクリーン座標推定
- ブラウザ内完結（映像データをサーバーに送信しない）
- キャリブレーション不要: 展示用途では粗い推定が POVM として物理的に正当化される
- カメラ拒否/不在時はマウス座標にフォールバック
- 出力: 正規化座標 (0-1, 0-1) を GazeTracker.onGaze コールバックで配信

### 5.4 エンタングルメント（Bell状態モデル）

タイミング同期 + 局所基底による独立結果:

```
Bell状態: |Φ+⟩ = (|00⟩+|11⟩)/√2

A が タイル[i] を観測
  → A のタイル[i] が A のパターン（例: 猫）で崩壊
  → WebSocket でタイルインデックス i をサーバーに送信
  → サーバーが B にリレー
  → B のタイル[i] が B のパターン（例: 眼）で崩壊

・崩壊タイミング = 相関（非局所的）
・崩壊結果 = 各ディスプレイの局所基底（パターン画像）で決定
・A と B は異なるパターンを表示（role-based offset）
```

- 二台のクライアントが一つのサーバーに WebSocket 接続
- サーバー（server.js）は崩壊イベントのリレーのみ。量子状態は管理しない
- Aの観客がタイル i を注視 → Aで崩壊 → Bでも同時に崩壊（ただしBの像はBのパターン）
- Bの観客は「自分が見ていない場所が確定し、しかも自分とは異なる像が出現した」ことを体験する

## 6. 展示設計

### 6.1 空間配置

- **二台のディスプレイ、同一空間**
- **推奨: 背中合わせ配置**
  - 通常の鑑賞では相手の画面が見えない
  - 不可解な挙動に気づいた観客が振り返ると相手の画面が見える
  - しかし同時に両方は見られない → 確認不可能性 ≈ 非局所的相関は信号伝達に使えないという性質
  - 相手の存在は気配として感じるが、画面は見えない

### 6.2 時間構造

- **物語性なし。定常状態で永続的に動作**
- 量子シミュレーションは常にサイクルを回し続ける
- 画像変容は非常にゆっくり進行（数分〜数十分スケール）
- 朝の観客と夕方の観客で異なる海を見る
- **観客不在時**: 測定がないため全てが重ね合わせに回帰 → 画面は均質な揺らぎ
- **観客到来時**: 注視によって画像が「現像」され始める

### 6.3 無音

- 視覚のみで完結
- 江渡 +/- Minimal VJ の禁欲的美学を踏襲

## 付録A: 主要な数学的定義

### 状態ベクトル

```
|ψ⟩ = Σ c_k |k⟩,  Σ |c_k|² = 1,  c_k ∈ C
```

### 縮約密度行列

```
ρ_i = Tr_{≠i}(|ψ⟩⟨ψ|)

ρ_i の要素:
  ⟨+|ρ_i|+⟩  → + になる確率（セルの「+ らしさ」）
  ⟨−|ρ_i|−⟩  → − になる確率
  |⟨+|ρ_i|−⟩| → コヒーレンス（量子性の度合い）
```

### フォン・ノイマンエントロピー

```
S(ρ_i) = −Tr(ρ_i log ρ_i)

S = 0    → 純粋状態（エンタングルなし、または全体を見ている）
S = ln2  → 最大混合状態（最大エンタングルメント）
```

### ベル状態

```
|Φ+⟩ = (|++⟩ + |−−⟩) / √2   ← 同相関
|Φ−⟩ = (|++⟩ − |−−⟩) / √2
|Ψ+⟩ = (|+−⟩ + |−+⟩) / √2   ← 反相関
|Ψ−⟩ = (|+−⟩ − |−+⟩) / √2
```

### 測定（POVM）

```
注視点 (x_0, y_0) からセル (x_i, y_i) への測定強度:

w_i = exp(−d_i² / 2σ²)

  d_i: 注視点からの距離
  σ: アイトラッキング精度に基づく分散

w_i ≈ 1 → 強い射影測定（ほぼ確定）
w_i ≈ 0 → 測定なし（状態不変）
0 < w_i < 1 → 部分測定（POVM）
```
